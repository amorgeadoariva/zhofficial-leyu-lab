# zhofficial-leyu-lab

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为 HTML 页面集合的存储与展示空间。

## 项目简介

本仓库旨在提供一种简洁的方式，将多个独立的 HTML 页面集中管理并发布。每个页面均可独立访问，适用于信息展示、工具页面、存档分享等场景。页面内容保持原样归档，不进行额外修改或重定向。

## 目录说明

```
zhofficial-leyu-lab/
├── index.html          # 仓库首页（可选）
├── page-1.html         # 独立页面示例
├── page-2.html         # 独立页面示例
└── assets/             # 页面共用资源（如图片、样式、脚本）
```

- `index.html`：仓库根目录下的首页，通常包含页面索引或说明。
- 各 `.html` 文件：独立发布的 HTML 页面，每个文件代表一个独立内容单元。
- `assets/`：存放页面所需的静态资源，可根据需要自行组织子目录。

## 页面归档说明

- 所有页面均为纯静态 HTML，不依赖后端服务。
- 页面内容在归档时保持原始状态，不进行内容审查或修改。
- 归档页面可能包含外部资源（如字体、图片、脚本），请确保资源链接有效或已本地化。
- 本仓库不保证页面内容长期可用，但会尽力维持文件完整性。

## 维护说明

- 欢迎通过 Pull Request 提交新的 HTML 页面，或对现有页面进行更新。
- 提交前请确保页面内容合规，不包含恶意代码或侵犯他人权益的信息。
- 如需删除或隐藏某个页面，请在仓库中直接操作或联系维护者。
- 本仓库由社区维护，不提供任何形式的担保或技术支持。
